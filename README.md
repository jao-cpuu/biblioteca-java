# biblioteca-java
Projeto desenvolvido com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO) utilizando a linguagem Java.

Este sistema simula uma biblioteca, permitindo representar diferentes tipos de itens do acervo, como livros, revistas e DVDs, além de usuários e funcionários.
O foco principal do projeto é aplicar conceitos fundamentais de POO na prática, com organização em pacotes e separação de responsabilidades.

O objetivo deste projeto é consolidar conhecimentos em Java e POO, servindo como base para o desenvolvimento de sistemas mais complexos no futuro.

Desenvolvido por João Pedro.

Este projeto é de uso educacional.
